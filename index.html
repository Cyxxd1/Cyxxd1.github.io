    BumpLess

    An IOS based application designed to work as a navigational aid for the visually impaired

    BumpLess is an IOS based application that is designed to aid the visually impaired navigate indoors.

    The app will scan the environment, accept user commands via buttons or voice, and project a path to help users navigate to specific locations or items using audio and haptic feedback.

    This intuitive tool will provide real-time, non-visual guidance, allowing users to confidently and safely navigate their homes.

    Maze Solver

    A Java based program that allows for users to test various algorithms and maze layouts.

    This Java-based Maze Loader is a versatile tool designed for exploring and testing maze-solving algorithms. It comes equipped with a set of default algorithms and pre-designed maze layouts, enabling users to see how standard methods—such as Depth-First Search, Breadth-First Search, and A*—perform in navigating complex mazes.

    In addition, the loader offers the flexibility for users to integrate their own custom algorithms and maze layouts. This modular design not only provides an educational platform for comparing algorithm performance but also encourages experimentation and innovation in maze-solving strategies.
